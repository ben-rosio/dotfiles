Dotfiles
========

Installation
------------
Dotfiles managed by [fresh](http://freshshell.com).

```bash
FRESH_LOCAL_SOURCE=ben-rosio/dotfiles bash -c "`curl -sL get.freshshell.com`"
```

Post Installation
-----------------
```bash
cd ~/.vim/bundle/YouCompleteMe/install.sh
cd ~/.vim/bundle/tern_for_vim/ && npm install
```
